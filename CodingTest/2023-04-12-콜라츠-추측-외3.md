# 1. 콜라츠 추측

## << 문제 내용

1937년 Collatz란 사람에 의해 제기된 이 추측은, 주어진 수가 1이 될 때까지 다음 작업을 반복하면, 모든 수를 1로 만들 수 있다는 추측입니다. 작업은 다음과 같습니다.

1-1. 입력된 수가 짝수라면 2로 나눕니다. 
1-2. 입력된 수가 홀수라면 3을 곱하고 1을 더합니다. 
2. 결과로 나온 수에 같은 작업을 1이 될 때까지 반복합니다. 
예를 들어, 주어진 수가 6이라면 6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1 이 되어 총 8번 만에 1이 됩니다. 위 작업을 몇 번이나 반복해야 하는지 반환하는 함수, solution을 완성해 주세요. 단, 주어진 수가 1인 경우에는 0을, 작업을 500번 반복할 때까지 1이 되지 않는다면 –1을 반환해 주세요.

## << 코드 작성

```python
class a:
    def __init__(self):
        self.x = None
        self.callcount = 0
    
    def even(self):
        self.callcount += 1
        self.x = self.x/2
    def odd(self):
        self.callcount += 1
        self.x =  (3*self.x +1)
    
b = a()

def solution(num):
    b.x = num
    while b.x != 1:
        if b.x % 2 == 0:
            b.even()
        elif b.x % 2 == 1:
            b.odd()
        
        if b.callcount == 500:
            b.callcount = -1
            break
    
    answer = b.callcount
    return answer
```

***

callcount 변수를 만들고 간편하게 증가시키고 싶어서 클래스를 만들었습니다.
클래스 내부에 있으면 메인 함수에서 귀찮게 callcount를 증가시키지 않아도 돼서... 는 결과적으로 귀찮은건 똑같았네   

## << 모범 답안

```python
def collatz(num):
    for i in range(500):
        num=num/2 if num%2==0 else num*3+1
        if num==1:
            return i+1
    return -1
```

예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

## << 코드 작성

```python
def solution(n):
    a = str(n)
    b = 0
    for i in range(len(a)):
        b += int(a[i])
    answer = b

    return b
```

# 3. 최댓값과 최솟값

## << 문제 내용

문자열 s에는 공백으로 구분된 숫자들이 저장되어 있습니다. str에 나타나는 숫자 중 최소값과 최대값을 찾아 이를 "(최소값) (최대값)"형태의 문자열을 반환하는 함수, solution을 완성하세요.
예를들어 s가 "1 2 3 4"라면 "1 4"를 리턴하고, "-1 -2 -3 -4"라면 "-4 -1"을 리턴하면 됩니다.

## << 코드 작성

```python
def solution(s):
    num_list = s.split()
    maximum, minimum = -100000, 100000
    for i in num_list:
        i_int = int(i)
        if i_int < minimum:
            minimum = i_int
        if i_int > maximum:
            maximum = i_int
        
    answer = str(minimum) + ' ' + str(maximum)
    return answer
```

문자열에 split()을 적용하면 자동으로 공백을 이용해 잘라 리스트로 변환하는데, 이를 int로 변환하여 max와 min을 찾습니다.  
C언어의 최대, 최소를 찾는 알고리즘이 익숙하여 이를 사용했지만, python에는 좋은 내장함수가 많다는 것을 잠시 잊고 있었습니다...  

## << 모범 답안

```python
def solution(s):
    s = list(map(int,s.split()))
    return str(min(s)) + " " + str(max(s))
```

# 4. 추억 점수

## << 문제 내용

사진들을 보며 추억에 젖어 있던 루는 사진별로 추억 점수를 매길려고 합니다. 사진 속에 나오는 인물의 그리움 점수를 모두 합산한 값이 해당 사진의 추억 점수가 됩니다. 예를 들어 사진 속 인물의 이름이 ["may", "kein", "kain"]이고 각 인물의 그리움 점수가 [5점, 10점, 1점]일 때 해당 사진의 추억 점수는 16(5 + 10 + 1)점이 됩니다. 다른 사진 속 인물의 이름이 ["kali", "mari", "don", "tony"]이고 ["kali", "mari", "don"]의 그리움 점수가 각각 [11점, 1점, 55점]]이고, "tony"는 그리움 점수가 없을 때, 이 사진의 추억 점수는 3명의 그리움 점수를 합한 67(11 + 1 + 55)점입니다.

그리워하는 사람의 이름을 담은 문자열 배열 name, 각 사람별 그리움 점수를 담은 정수 배열 yearning, 각 사진에 찍힌 인물의 이름을 담은 이차원 문자열 배열 photo가 매개변수로 주어질 때, 사진들의 추억 점수를 photo에 주어진 순서대로 배열에 담아 return하는 solution 함수를 완성해주세요.

## << 코드 작성

```python
def solution(name, yearning, photo):
    answer = []
    score_dict = {}
    for index, name in enumerate(name):
        score_dict[name] = yearning[index]
    
    score = 0
    
    for a_photo in photo:
        for name in a_photo:
            score += score_dict.get(name,0)
        answer.append(score)
        score = 0
        
    return answer
```

역시 사람 이름과 추억 점수를 딕셔너리에 저장하는 게 빠를 것이라고 생각했습니다.  
후에 score 변수에 추억점수를 합산해 구한 뒤, answer에 append()를 이용하여 추가합니다.
